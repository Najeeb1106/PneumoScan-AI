About the Project

PneumoScan AI was developed as an advanced, end-to-end deep learning project aimed at assisting medical professionals in the rapid diagnosis of pneumonia using chest X-ray imagery.

Dataset & Architecture

Utilizing a verified Kaggle dataset consisting of over 5,800 pediatric chest X-rays, we designed and trained a custom 3-layer Convolutional Neural Network (CNN) architecture using TensorFlow and Keras. The model incorporates techniques like MaxPooling, Dropout, and Data Augmentation to prevent overfitting and improve generalization.

The Importance of Recall

In medical diagnostics, false negatives (failing to identify an actual disease) carry far more severe consequences than false positives. Thus, our model was strictly optimized for recall (sensitivity). By achieving a 98% recall rate on test data, the system virtually guarantees that pathological indicators of pneumonia are flagged for review by a professional.
